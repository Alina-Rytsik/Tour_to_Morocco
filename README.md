# 🐫 Tour to Morocco – Vanilla JS Landing Page

A highly atmospheric, high-performance landing page for a travel agency specializing in expeditions to Morocco. 

[🔗 Live Demo](https://alina-rytsik.github.io/Tour_to_Morocco/)

## 🎯 Project Goals
The main purpose of this project was to demonstrate core Frontend development skills without using heavy frameworks:
- Advanced DOM manipulation.
- Asynchronous requests (Fetch API).
- Custom UI component logic (Slider, Tabs, Timer).
- Modular CSS architecture with SCSS.

## 🚀 Key Features
- **Dynamic Countdown Timer:** Real-time discount timer (48h) with LocalStorage persistence.
- **Custom Slider & Tabs:** Hand-coded navigation and gallery logic.
- **Expedition Calculator:** Interactive price calculation based on destination, days, and group size.
- **Form Integration:** Functional contact form using Fetch API and Formspree (delivers emails directly to the owner).
- **Smooth Animations:** Integrated "fade" and "splash" effects for enhanced UX.

## 🛠 Tech Stack
- **JavaScript (ES6+):** Modular approach, Fetch API, Promises, LocalStorage.
- **SCSS:** BEM methodology, variables, mixins, and nested architecture.
- **HTML5:** Semantic markup.
- **Tools:** Live Sass Compiler, GitHub Pages.
